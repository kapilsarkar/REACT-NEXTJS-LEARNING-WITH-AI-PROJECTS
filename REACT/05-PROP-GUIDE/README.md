# React Prop Guide

A complete practical guide to understanding **Props in React** through reusable examples.

This project covers the core concepts of props and component communication in React.

---

## Topics Covered

### 1. Basic Props

Learn how to pass simple data from parent to child components.

Examples:

* Strings
* Numbers
* Booleans

---

### 2. Children Props

Learn how to pass JSX elements or components inside custom components.

Helps understand:

* Component composition
* Reusable wrappers
* Layout components

---

### 3. Complex / Nested Props

Learn how to pass:

* Objects
* Arrays
* Functions
* Nested data structures

Helps build real-world reusable UI.

---

### 4. Ref Props

Learn how to pass refs to child components using:

* `useRef`
* `forwardRef`

Used for:

* Focus management
* DOM access
* Input control

---

### 5. Theme Props + Context API

Learn how to combine:

* Props
* Context API
* Shared global state

Used for:

* Theme toggling
* Global UI state
* Avoiding prop drilling
