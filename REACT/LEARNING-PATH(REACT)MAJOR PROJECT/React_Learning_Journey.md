# React Learning Journey

## Goal

Become confident with advanced React before moving to Next.js.

## Learning Sequence

### 1. React Hook Form + Zod

- Form handling
- Validation
- Error messages
- Reusable form components

**Mini Practice** - Login Form - Registration Form - Contact Form

------------------------------------------------------------------------

### 2. React Router

- BrowserRouter
- Routes & Route
- Nested Routes
- Dynamic Routes
- Route Parameters
- Protected Routes
- Layout Routes
- 404 Page

**Mini Practice** - Multi-page dashboard

------------------------------------------------------------------------

### 3. Redux Toolkit

- Store
- Slice
- configureStore
- useSelector
- useDispatch
- Async Thunks
- Project Structure

**Mini Practice** - Shopping Cart - Authentication State

------------------------------------------------------------------------

### 4. useRef

- DOM access
- Focus input
- Timers
- Previous values

------------------------------------------------------------------------

### 5. Custom Hooks

- Reusable logic
- Data fetching hooks
- Form hooks

------------------------------------------------------------------------

### 6. useMemo & useCallback

- Performance optimization
- Memoization
- Prevent unnecessary re-renders

------------------------------------------------------------------------

### 7. Lazy Loading

- React.lazy()
- Suspense
- Code splitting
- Route-based lazy loading

------------------------------------------------------------------------

### 8. Appwrite

- Authentication
- Database
- Storage
- CRUD Operations
- Deployment

------------------------------------------------------------------------

### 9. Zustand Revision

Use Zustand inside the final project wherever appropriate.

------------------------------------------------------------------------

## Final Milestone

Build one production-style React application that combines everything
above.

After completing the project, begin learning Next.js.
