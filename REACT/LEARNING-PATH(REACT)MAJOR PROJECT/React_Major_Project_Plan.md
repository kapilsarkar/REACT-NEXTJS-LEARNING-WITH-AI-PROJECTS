# Major React Project

## Employee Management System

## Objective

Build an enterprise-style Employee Management System using modern React
practices.

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- React Hook Form
- Zod
- Redux Toolkit
- Appwrite
- Zustand (optional)
- React Toastify

------------------------------------------------------------------------
